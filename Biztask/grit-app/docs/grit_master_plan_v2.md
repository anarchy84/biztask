# 🚀 사업자 전용 SNS [GRIT] 프로젝트 기획서 (V2.0)

## 1. 프로젝트 개요
- **서비스명:** GRIT (그릿)
- **슬로건:** 사장님들의 독한 성공, 그릿에서 시작하다.
- **핵심 가치:** - **신뢰(Trust):** API 기반 사업자 인증으로 유령 회원 차단.
  - **익명(Anonymity):** 사업자 뒤에 숨은 한 개인으로서의 솔직한 소통.
  - **연결(Connection):** 업종별 시크릿 매칭 및 팔로우 기반 비즈니스 네트워크.
- **플랫폼:** 하이브리드 앱(iOS/Android) + PC 웹버전.

---

## 2. 주요 기능 상세 (Feature List)

### 2.1 회원가입 및 인증 시스템
| 기능 | 상세 내용 |
| :--- | :--- |
| **사업자 인증** | 국세청 API 등을 활용한 실시간 사업자 번호 검증 (업종명 자동 추출). |
| **익명성 보장** | 인증은 받되, 활동은 랜덤 닉네임 혹은 설정한 닉네임으로 진행 (블라인드 방식). |
| **휴대폰 인증** | 본인 확인용 (현재는 기획안에만 포함, 추후 연동). |
| **유료 회원 (Blue)** | 월 구독료 지불 시 '인증마크(파란딱지)' 부여 및 프리미엄 기능 오픈. |
| **탈퇴/재가입 정책** | - 가입된 사업자 번호는 1인 1계정 원칙.<br>- 탈퇴 후 7일간 재가입 불가.<br>- 탈퇴 후 30일 이내 재가입 시 과거 데이터 복구 서비스 제공. |

### 2.2 커뮤니티 및 피드 (UI/UX)
- **UI 베이스:** 인스타그램의 비주얼 피드 + 스레드의 스레드형 댓글 구조.
- **팔로우 시스템:** - 특정 사업자(닉네임) 팔로우 기능.
  - 팔로잉/팔로워 리스트 관리.
- **피드 정렬 및 알고리즘:** - **추천순 (Default):** 일론 머스크의 X(구 트위터) 알고리즘 베이스. 
    - **발견(Discovery):** 사용자의 관심사 기반 새로운 콘텐츠 추천.
    - **네트워크(Follower):** 팔로잉 중인 사용자의 최신 글이 일정 비율(예: 30-40%)로 믹스되어 노출.
  - **최신순:** 실시간 업로드 순.
- **포스트 타입:** 일반 글, 이미지, 인용하기(X 방식), 추천(좋아요).
- **카테고리:** 어드민에서 관리하는 업종별/세부 주제별 커뮤니티 섹션.

### 2.3 시크릿 기능 (Private & Matching)
- **비공개 게시판:** 인증된 사업자만 진입 가능한 고밀도 정보 공유의 장.
- **매칭/협업:** 업종 간 제휴나 협업 제안을 할 수 있는 전용 탭.
- **구인구직 (유료):** 검증된 사장님들이 올리는 구인 공고 (유료 회원 전용).

### 2.4 메시징 (Social)
- **1:1 DM:** 회원 간 자유로운 다이렉트 메시지. (팔로우 관계에 따른 수신 제한 설정 가능)
- **그룹 채팅 (단톡):** **유료 회원만 개설 가능.** (특정 주제나 지역 모임 주도).

---

## 3. 수익 모델 (Monetization Strategy)

1. **프리미엄 구독 (Member):** - 파란딱지(인증마크) 부여.
    - 단톡방 개설 권한.
    - 구인구직 포스팅 권한.
    - 광고 제거(Optional).
2. **광고 수익 (Ads):** - 피드 내 네이티브 광고 (인스타 방식).
    - 구글 애드센스 및 메타 네트워크 광고 연동.
3. **비즈니스 수수료:** - 시크릿 매칭/협업 성사 시 발생하는 중개 수수료 (추후 확장).

---

## 4. 기술 스택 및 아키텍처 제안

- **App Framework:** **Flutter** 또는 **React Native** (하이브리드 앱 지향).
- **Backend:** **Supabase** (PostgreSQL + Auth + Realtime).
- **PC Web:** React 또는 Next.js를 활용한 반응형 웹.
- **API 연동:** 공공데이터포털(사업자등록정보 진위확인 API).

---

## 5. 어드민 대시보드 (Admin Panel)

- **회원 관리:** 인증 대기 회원 승인 및 블랙리스트 관리.
- **콘텐츠 관리:** 팔로우/추천 알고리즘 가중치 조정 및 카테고리 관리.
- **광고 관리:** 인앱 피드 광고 이미지 및 링크 관리.
- **통계:** DAU, 팔로우 전환율, 매출 현황 트래킹.

---

## 6. 향후 로드맵
- **Step 1:** MVP 개발 (사업자 인증 + 익명 피드 + 팔로우 기능).
- **Step 2:** 추천 알고리즘 고도화 (팔로워 콘텐츠 믹싱 로직 적용).
- **Step 3:** 유료 결제 모듈 및 인증마크 기능 도입.
- **Step 4:** 시크릿 매칭 및 구인구직 기능 고도화.
